# safedesk-stories
Stories generation

## Requirements
- Python 3
- pip3

## Dependencies

- Faker, used to generate fake data

```sh
$ pip install faker
$ pip install graphviz
```
or
```sh
$ pip install -r requirements.txt
```


## Usage

Usefull command

```sh
$ python3 [script.py]
$ python3 init.py [FolderToAnalyse] [Output]
```
